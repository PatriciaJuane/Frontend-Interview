# Coding question
Implement a Typeahead (Autocomplete) component that fetches and displays word suggestions as the user types.


Requirement
- As the user types, fetch and display matching suggestions from the API given below.
- If no results are found, display No results found.
- If Api fails display - Something went wrong.
- The solution should be optimized.


Endpoint - https://api.datamuse.com/sug?s={query}


Request:
https://api.datamuse.com/sug?s=car

Response:

[
  { "word": "car", "score": 31247 },
  { "word": "card", "score": 28620 },
  { "word": "care", "score": 27760 }
]


Bonus
- Allow clearing the input using an "X" icon or clear button.
- Add keyboard navigation

Constraints
- Dont use any third party library like loadsh ect
- use of AI tool is prohbited
